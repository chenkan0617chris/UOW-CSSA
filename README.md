# 伍伦贡大学中国学生学者联合会
University of Wollongong - Chinese Students Scholars Association

欢迎大家加入UOW中国学联！

伍伦贡中国学生学者联合会，简称UOW中国学联或UOWCSSA，是旨在帮助中国留学生的组织。UOWCSSA现已在中国驻悉尼总领馆注册，为在伍伦贡生活的学生及学者提供友好交流的平台，也为学生联系领馆搭建桥梁.
